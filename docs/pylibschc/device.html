pylibschc.device¶

User-facing representation of libSCHC devices.

class pylibschc.device.Device(device_id: int, mtu: int, duty_cycle_ms: int)[source]¶

Bases: object

A libSCHC device.

This wraps pylibschc.libschc.Device for a more pythonic usage. Two devices with the same device_id are effectively the same object (i.e., this class is a multiton)

Parameters:

  • device_id – The libSCHC-internal identifier of the device.

  • mtu – The maximum transmission unit of the link layer of the device.

  • duty_cycle_ms – The duty cycle in milliseconds of the device.

mtu: pylibschc.device.Device¶

The maximum transmission unit of the link layer of the device.

duty_cycle_ms: pylibschc.libschc.Direction¶

The duty cycle in milliseconds of the device.

property compression_rules: List[CompressionRule]¶

The compression rules of this device.

If they are set to None, the internal compression context of the libSCHC version of this device will be free’d and its rule count set to 0. If it is set to a list of compression rules, the context will be updated accordingly.
