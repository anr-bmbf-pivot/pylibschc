class pylibschc.libschc.BitArray¶

Bases: object

A bit-based array. Wraps the schc_bitarray_t type of libSCHC.

Parameters:

n (typing.Union[bytes, int]) – Input to the BitArray. If the type of n is bytes those bytes will be copied into BitArray.buffer, the maximum size then is the length of n. If the type of n is int, an empty BitArray of maximum size n is created.

Raises:

MemoryError – if internal memory for BitArray.buffer can not be allocated.

bit_length¶

Length of BitArray.buffer in bits.

buffer¶
Type:

bytes

The underlying bytes buffer of the bit array.

copy_bits()¶

Copy bits to BitArray.buffer.

Parameters:

  • pos (int) – Position of the first bit in BitArray.buffer. Must be at least 0 and not exceed BitArray.bit_length.

  • data (bytes) – Data to copy to BitArray.buffer.

  • length (int) – Number of bits to copy from data to BitArray.buffer. Must be at least 0 and not exceed BitArray.bit_length.

Raises:

  • ValueError – If length is lesser than 0.

  • ValueError – If pos is lesser than 0.

  • ValueError – If pos + length is greater than BitArray.bit_length.

get_bits()¶

Get at a maximum 32 bits from the BitArray as integer.

Parameters:

  • pos (int) – Position of the first bit in BitArray.buffer. Must be at least 0 and not exceed BitArray.bit_length.

  • length (int) – Number of bits to read from BitArray.buffer as of pos. Must be between 0 and 32.

Raises:

  • ValueError – If length is lesser than 0 or greater than 32.

  • ValueError – If pos is lesser than 0.

  • ValueError – If pos + length is greater than BitArray.bit_length.

Returns:

the bits from pos to pos + length (not inclusive) in BitArray.buffer converted to an integer.

Return type:

int

length¶

Length of BitArray.buffer in bytes.

If BitArray.bit_length overflows a byte, this value will be (BitArray.bit_length // 8) + 1, otherwise it will be BitArray.bit_length // 8.

offset¶

Offset in bits within BitArray.buffer.

padding¶

Padding in bits after BitArray.buffer.

size¶

Maximum size in bytes of the memory used for BitArray.buffer.

class pylibschc.libschc.FragmentationConnection¶

Bases: object

A fragmentation connection. Wraps the schc_fragmentation_t type of libSCHC.

Parameters:

ops (FragmenterOps) – Operation callbacks.

bit_arr¶
Type:

BitArray

The BitArray of the fragmentation connection if initialized, None otherwise. Wraps the bit_arr member of schc_fragmentation_t.

fragment()¶

Send FragmentationConnection.bit_arr, if necessary fragmented. Wraps schc_fragment().

Raises:

MemoryError –

  • When no schc_mbuf_t in its head member could not be allocated,

  • if no fragmentaftion rule for the initialized mode can be found, or

  • if the state of the fragmentation connection is invalid.

Retval NO_FRAGMENTATION:

If the packet was not fragmented.

Retval SUCCESS:

If the packet was fragmented.

Return type:

FragmentationResult

fragmented¶
Type:

bool

True after calling FragmentationConnection.fragment() when the received packet is a fragment, False if not.

init_rx()¶

Initialize connection for reception.

Parameters:

  • device_id (int) – The identifier of the device this connection is to be used with.

  • bit_arr (BitArray) – The BitArray for the connection.

  • dc – Duty cycle in milliseconds for the connection.

init_tx()¶

Initialize connection for transmission.

Parameters:

  • device_id (int) – The identifier of the device this connection is to be used with.

  • bit_arr (BitArray) – The BitArray for the connection.

  • mtu – The link layer MTU for this connection in bytes.

  • dc – Duty cycle in milliseconds for the connection.

  • mode (FragmentationMode) – The fragmentation mode for this connection.

input()¶

Must be called whenever a packet is received. Wraps schc_input().

When the packet was a fragment, FragmentationConnection.fragmented will be True, otherwise it will be False.

Parameters:

buffer (BitArray) – The received packet.

Raises:

MemoryError – If a new connection object or one of its members could not be allocated.

Returns:

This connection, if buffer was an ACK, a new FragmentationConnection, when it was a fragment. FragmentationConnection.reassemble() shall be called in the latter case to initialize or continue reassembly.

is_conn()¶
mbuf¶
Type:

bytes

The reassembly buffer of the fragmentation connection. Wraps and combines the functionality of get_mbuf_len() and mbuf_copy()

reassemble()¶

Handle the last received fragment. Wraps schc_reassemble().

Retval ONGOING:

If reassembly is still missing fragments.

Retval COMPLETED:

If the fragment handled was the last missing fragment.

Retval STAY_ALIVE:

If reassembly was completed, but the connection still is kept open, e.g., in case another ACK needs to be sent.

Return type:

ReassemblyStatus

Reset a connection.
