Validator to check if CompressionRuleField.MO_param_length is lesser or equal to CompressionRuleField.field_length when CompressionRuleField.MO is pylibschc.libschc.MO.MSB or CompressionRuleField.action is pylibschc.libschc.CDA.LSB

Parameters:

  • values – name-to-value mapping of previously validated fields
